문제 · 순환오류

셀별 fit이 자기참조로 무너진다

셀 하나하나에 물리 파라미터를 역산해 수명을 예측하면 추정과 목표가 서로를 참조하는 순환(circularity)에 빠진다. 같은 분포 안에서는 정확해 보여도, 새 제조 배치·공급사·급속충전 프로토콜로 옮기면 성능이 붕괴한다.

→
해결 · 분포 경계 + 연산자 학습

외부 분포로 경계를 학습하고 궤적을 직접 예측

개별 셀 fit 대신 공개 데이터의 분포로 물리 파라미터의 유효 경계를 학습하고, 사이클→열화 연산자(DeepONet)로 용량 궤적을 직접 예측한다. 분포 밖 입력은 외삽하지 않고 flag(abstain)해 조용한 오탐을 막는다.

문제

배터리 수명을 실측하려면 수백~수천 사이클을 돌려야 하고, 물리 시뮬레이션은 셀 하나당 수 분이 걸려 대규모 탐색에 부담이 크다. 초기 몇십 사이클만으로 수명과 열화를 신뢰성 있게 예측할 수 있으면 개발 주기를 크게 줄일 수 있다.

접근

  • 물리 + 데이터 결합. PyBaMM SPM 전기화학 모델을 기반으로 삼고, 무거운 시뮬레이션을 DeepONet(Operator Learning) 신경망으로 대체해 추론을 가속했다.
  • 초기 사이클 기반 예측. 초기 사이클의 특징만으로 수명(EOL)과 열화 궤적을 예측하도록 구성했다.
  • 교차 데이터셋 검증. TRI, HUST, CALCE 공개 데이터셋에서 셀 분리(cell-disjoint) 방식으로 평가해 과적합을 통제했다.
  • 도구화. 실험 프로토콜 생성을 드래그앤드롭 GUI 도구로 제품화해 공개 저장소로 배포했다.

결과

초기 사이클만으로 셀 분리(cell-disjoint) 조건에서 수명을 예측하고, 물리 시뮬레이션 대비 추론을 대폭 가속했다. 공개 데이터셋(TRI·HUST·CALCE) 기준 결과는 다음과 같다.
