파일 접근 제어 규칙 자동 생성 시스템

엔드포인트에서 파일 접근을 후킹해, 랜섬웨어와 데이터 유출을 막는 접근 제어 규칙을 ML로 자동 생성하는 시스템이다. 광운대학교와 테르텐의 산학협력으로 진행했다.

문제

파일 접근 제어 규칙을 사람이 일일이 정의하면 커버리지가 낮고 유지가 어렵다. 랜섬웨어와 내부 데이터 유출을 막으려면 실제 접근 패턴에서 규칙을 자동으로 학습해야 한다.

접근

  • API 후킹. MS Detours로 파일 접근 API를 후킹해 엔드포인트에서 접근 이벤트를 수집했다.
  • ML 규칙 생성. word2vec으로 접근 맥락을 임베딩하고 의사결정트리로 패턴을 학습해, NIST ABAC(속성 기반 접근 제어) 규칙을 자동 생성했다.
  • 화이트리스트 기반 방어. 학습된 규칙으로 정상 접근만 허용하는 화이트리스트를 구성해 랜섬웨어·유출을 차단했다.
