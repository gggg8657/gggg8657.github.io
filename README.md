# 김동주 — AI Scientist Portfolio

공개 포트폴리오 사이트. **Live: https://gggg8657.github.io**

한국원자력연구원(KAERI) 인공지능응용연구실 석사후연수생. 지능형 CCTV(VAD·VLM·Agentic), 방사성의약품 SSTR2 in silico, Robot AI, 배터리 Physics-Informed ML.

## 구조

정적 멀티페이지 사이트. 빌드 도구·프레임워크 없음, GitHub Pages가 root에서 서빙.

- `index.html` — 랜딩(허브). 프로젝트 카드 클릭 → 상세페이지로 이동.
- `projects/*.html` — 프로젝트 상세페이지 (cctv, sstr2, audisp, battery, file-access).
- `style.css` — 공용 스타일 (라이트/다크 대응).
- `main.js` — 스크롤 리빌 (prefers-reduced-motion 존중).

## 공개 / 대외비 경계

이 저장소는 **공개 가능한 내용만** 담는다. 상세 원본과 대외비 자료는 별도 비공개 저장소(`career-applications`)에서 관리한다.

| 공개 (이 저장소) | 대외비 (비공개) |
|---|---|
| 시스템 아키텍처 개념도 | SSTR2 게이트 임계값, 실제 후보 구조·서열 |
| 학회 발표·공개 저장소로 검증된 수치 | 내부 과제 정량 결과, private 레포 유래 수치 |
| 공개 저장소 링크, 수상, 학력, 스킬 | SPARC·EMP 시설 세부 |
| 업무 이메일, 전문 프로필 링크 | 개인 연락처(전화·주소·생년월일) |

새 내용을 추가할 때는 위 경계를 먼저 확인한다. 애매하면 비공개로 둔다.
